# jobfill-extension
JobFill Extension Application Home

Installation: 
Step 1 — Unzip
  
  Right-click on the .zip file and extract it to a folder which you'll leave in place (don't delete it after — Chrome needs the folder to stay where it is).

Step 2 — Open Chrome Extensions
  In Chrome, go to the address bar and type:
  chrome://extensions
  then press Enter.

Step 3 — Enable Developer Mode
  In the top-right corner of the Extensions page, toggle Developer mode ON. A new row of buttons will appear.

Step 4 — Load Unpacked
  Click Load unpacked, then navigate to and select the ext42 folder (the folder itself, not the zip). Click Select/Open.

Step 5 — Pin it to the toolbar
  You'll see JobFill appear in your extensions list. To pin it to the toolbar so the ⚡ icon is always visible:
     -> Click the puzzle piece icon 🧩 in the top-right of Chrome
     -> Find JobFill in the list
     -> Click the pin icon 📌 next to it

The JobFill icon will now appear permanently in your toolbar. Click it to open the popup.
